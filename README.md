# Bookstore Management System

A project designed to manage a bookstore's inventory, sales, and customer data. The project integrates **SSIS**, **Power BI**, and **SQL** for data management, reporting, and analytics.

## Features
- ETL processes to extract, transform, and load sales and inventory data using SSIS.
- Analytical dashboards and reports created with Power BI.
- SQL database for storing books, sales transactions, and customer information.
- Data warehouse design for efficient reporting and analysis.

## Technologies Used
- **SSIS (SQL Server Integration Services)**
- **Power BI**
- **SQL Server**
- **Git** for version control

## Project Structure
/SSIS/ # Contains all ETL packages for extracting, transforming, and loading bookstore data
/PowerBI/ # Contains Power BI dashboards and reports (.pbix files) for analytics and visualization
/SQL/ # Contains SQL scripts to create databases, tables, and initial data


## Usage
1. Run SSIS packages to load and transform bookstore data.
2. Execute SQL scripts to create the database and tables.
3. Open Power BI reports (.pbix files) to visualize sales, inventory, and customer insights.
